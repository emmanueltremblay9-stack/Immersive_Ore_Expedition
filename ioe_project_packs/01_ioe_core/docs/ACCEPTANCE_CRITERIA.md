# Acceptance Criteria — Immersive Ore Expedition: Core

## General
- Project opens as a NeoForge 1.21.1 Java 21 project or can be migrated into the user's existing template.
- Module does not exceed its documented scope.
- Config is server-authoritative where relevant.
- Missing optional dependencies do not crash the game.
- Missing resources are skipped and logged.

## Resource safety
- No fake resources are generated.
- Excluded resources do not appear in province/config defaults.

## Evidence required from Codex
- Build result.
- Run/client or server smoke result when feasible.
- List of files changed.
- Config examples.
- Known limitations.
