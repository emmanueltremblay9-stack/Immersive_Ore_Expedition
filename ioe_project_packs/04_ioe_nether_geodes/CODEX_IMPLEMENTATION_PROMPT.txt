# Project Spec — IOE Nether Geodes

## Responsibility
This module turns Nether crystal resources into dangerous sub-lava geode expeditions.

## Must implement
- Giant Nether lava lake anchor detection.
- Geode placement only below valid large lava bodies.
- Nether Quartz geode growth/resource chambers.
- Extremely rare Ancient Debris hearts if configured and resource exists.
- Nether clue structures near lava shore/edges.

## Must not implement
- No random geodes in normal netherrack.
- No ancient debris spam.
- No non-whitelisted resource types.

## Design rule
Nether lava lakes become resource landmarks.
